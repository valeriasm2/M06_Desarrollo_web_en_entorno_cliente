Ejercicio 4

Enunciado: ¿Cuándo usar un preventDefault() en formularios?

Usar e.preventDefault() para evitar envío automático y validar primero.

Ejercicio 5

Enunciado: Explica paso a paso qué ocurre en este código:

setInterval(() => {contador++}, 1000);
Recuerda que para detenerlo usamos clearInterval(id) con el id devuelto por setInterval.
